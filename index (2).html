Installation

WebArsenal requires Node.js v18 or higher. Clone the repo and install dependencies.

$ git clone https://github.com/edwinnyandika/webarsenal.git
$ cd webarsenal
$ npm install
Puppeteer and Playwright download browser binaries (~300MB) on first install. Expected and required.

Quick Start

Mirror a full site

node core/super-mirror.js --url https://example.com --depth 4

Scrape a React SPA

node scrapers/spa-scraper.js --url https://app.example.com --wait-for '#root'

Run a full SEO audit

node analyzers/seo-auditor.js --url https://example.com --output report.json

Export data to SQLite

node exporters/to-sqlite.js --input data.json --db scrape.db

Push to AWS S3

node integrations/aws-s3-uploader.js --dir ./site_data --bucket my-bucket

Full Orchestration Example

Chain scripts into a complete extraction pipeline:

# Step 1: Bypass Cloudflare
node auth-helpers/cf-clearance-puller.js --url https://store.com

# Step 2: Deep mirror with cookie auth
node core/super-mirror.js --url https://store.com --depth 3 --cookie-jar cf_cookies.json

# Step 3: Upload to S3
node integrations/aws-s3-uploader.js --dir ./super-mirrored-site --bucket website-backups

# Step 4: Alert team on Slack
node integrations/slack-alerter.js --webhook YOUR_WEBHOOK --message "Mirror complete ✓"

About the Project

Built for devs
who move fast.

WebArsenal is the toolkit the web developer community needed but didn't have — 110 focused, composable modules covering every phase of web data extraction.

The problem
it solves

Web extraction is fragmented. Developers stitch together Puppeteer snippets, Cheerio tutorials, half-finished proxy scripts, and manual Postman exports just to get data out of a site they own.

There was no single, well-organized toolkit that handles the full pipeline — from "clone this site" through to "push results to Notion." WebArsenal solves that.

  • Self-contained scripts — one job per file, no hidden state
  • CLI-first — all flags, stdout output, pipe anywhere
  • Composable — output of one is natural input to the next
  • Zero SaaS — no API keys, no dashboards, no subscriptions
  • MIT licensed — use in commercial products freely

Who it's for

Full-stack devs, SEO engineers, data engineers, indie hackers, and security researchers (on systems they're authorized to test) all have something in this toolkit.

It's the kind of repo you bookmark and actually use — not a framework to learn, just scripts that work.
